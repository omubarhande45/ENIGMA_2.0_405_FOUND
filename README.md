# BreathSafe – AI Powered Lung Health & AQI Intelligence System

BreathSafe is an AI-driven respiratory health monitoring platform that combines **real-time air quality data** with **lung capacity analysis** to provide personalized health insights, preventive alerts, and breathing recommendations.

Built for hackathons and smart healthcare innovation, PulmoSense demonstrates how environmental data + health analytics can predict respiratory risk and improve lung wellness.

---

 Features

# 1. Lung Capacity Analyzer (Spirometry AI)

* Input: FEV1, FVC, PEF values
* Calculates lung function score
* Detects:

  * Normal lung function
  * Mild obstruction
  * Severe respiratory risk
* Generates AI-based health recommendations


# 2. Real-Time AQI Monitoring

Fetches live air quality data with:

* AQI value
* PM2.5 & PM10
* Health category
* Location-based pollution data


 # 3. AI Health Recommendation Engine

Combines:

* Lung capacity
* AQI pollution exposure
* Risk category

Provides:

* Personalized health alerts
* Mask recommendations
* Outdoor activity warnings
* Preventive care 

# 4. Breathing Exercise Recommendation

Suggests exercises based on lung health:

* Box breathing
* Deep breathing
* Recovery breathing
* Stress relief breathing
